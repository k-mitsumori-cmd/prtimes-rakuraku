# 🔍 Cursorでソース管理を見つける方法

「ソース管理」アイコンが見つからない場合の、複数の見つけ方を説明します。

## 方法1: ショートカットキー（最も簡単！）

### Macの場合
```
Cmd + Shift + G
```
（コマンド + シフト + G を同時に押す）

### Windowsの場合
```
Ctrl + Shift + G
```
（コントロール + シフト + G を同時に押す）

**これが一番簡単です！**

## 方法2: 左側のサイドバーから探す

### アイコンの場所

Cursorの**左側の縦に並んだアイコン**の中にあります。

通常、以下のような順番で並んでいます（上から下へ）：

1. 📁 **エクスプローラー**（フォルダアイコン）
2. 🔍 **検索**（虫眼鏡アイコン）
3. 🌿 **ソース管理**（分岐マーク・Gitアイコン）
4. 🐛 **実行とデバッグ**
5. 📦 **拡張機能**

### ソース管理アイコンの見た目

- **分岐マーク**（Y字型のアイコン）
- **Gitのロゴ**に似たアイコン
- **3つの丸がつながったアイコン**

### 見つけ方

1. Cursorの**左端**を見る
2. **縦に並んだアイコン**を探す
3. **分岐マーク**や**Gitっぽいアイコン**を探す
4. クリックする

## 方法3: メニューから開く

1. 画面上部のメニューバーを確認
2. 「**表示**」（View）メニューをクリック
3. 「**ソース管理**」または「**Source Control**」を選択

## 方法4: コマンドパレットから開く

1. `Cmd + Shift + P`（Mac）または `Ctrl + Shift + P`（Windows）を押す
2. 「**Source Control**」または「**ソース管理**」と入力
3. 「**View: Show Source Control**」を選択

## 方法5: ステータスバーから確認

Cursorの**画面下部**（ステータスバー）に、Gitの状態が表示されている場合があります。

- ブランチ名（例: `main`）
- 変更数（例: `3 changes`）

これをクリックしてもソース管理パネルが開きます。

## 🎯 最も簡単な方法（推奨）

**ショートカットキーを使うのが一番簡単です！**

### Mac
```
Cmd + Shift + G
```

### Windows
```
Ctrl + Shift + G
```

このキーを押すだけで、ソース管理パネルが開きます。

## 📸 見つからない場合

もし見つからない場合は：

1. **ショートカットキーを使う**（`Cmd + Shift + G`）
2. **コマンドパレットを使う**（`Cmd + Shift + P` → "Source Control"と入力）

これらは確実に開けます！

## ✅ 開けたら

ソース管理パネルが開くと、以下のような表示になります：

- **変更**セクション: 変更されたファイルの一覧
- **メッセージ**欄: コミットメッセージを入力する場所
- **✓ コミット**ボタン: コミットするボタン
- **...**ボタン: その他の操作（プッシュ、プルなど）

## 💡 ヒント

- ショートカットキー `Cmd + Shift + G` を覚えると、いつでもすぐに開けます
- 一度開くと、左側のアイコンがハイライトされるので、次回から見つけやすくなります
