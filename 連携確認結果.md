## 🎯 次のステップ

### 連携が完了していない場合

1. **Personal Access Tokenを作成**
   - GitHub → Settings → Developer settings → Personal access tokens
   - `repo`スコープを選択
   - トークンをコピー

2. **プッシュ時に認証**
   - プッシュ時にトークンを入力
   - 一度認証すれば次回から自動

### 連携が完了している場合
