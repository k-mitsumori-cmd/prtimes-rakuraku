# ⚡ 超簡単！CursorでGit操作（3ステップ）

「ソース管理」アイコンが見つからない？大丈夫！**ショートカットキー**を使えば一発で開けます！

## 🚀 超簡単3ステップ

### ステップ1: ソース管理を開く

**Mac**: `Cmd + Shift + G` を押す  
**Windows**: `Ctrl + Shift + G` を押す

これだけでソース管理パネルが開きます！

### ステップ2: ファイルを追加してコミット

1. 開いたパネルで「**すべての変更をステージング**」をクリック
   - または、各ファイルの横の「**+**」をクリック

2. 上部の「**メッセージ**」欄に説明を入力
   - 例: "ファイルを更新"
   - 例: "Vercel設定を追加"

3. 「**✓ コミット**」ボタンをクリック
   - または、`Cmd + Enter`（Mac）を押す

### ステップ3: GitHubにアップロード

1. パネル下部の「**...**」（3つの点）をクリック
2. 「**プッシュ**」を選択

## 🎯 これで完了！

たったこれだけです！

## 💡 覚えておくと便利

- **ソース管理を開く**: `Cmd + Shift + G`（Mac）または `Ctrl + Shift + G`（Windows）
- **コミット**: `Cmd + Enter`（Mac）
- **コミットしてプッシュ**: `Cmd + K, Cmd + Enter`（Mac）

## ❓ エラーが出た場合

### 認証エラー

プッシュ時に認証を求められたら：
1. Personal Access Tokenを作成（GitHubの設定から）
2. トークンを入力

詳細は `CursorとGitHub連携手順.md` を参照してください。

## 🎉 簡単でしょ？

ショートカットキー `Cmd + Shift + G` を覚えるだけで、Git操作が超簡単になります！
